    如何指定 Hugo 网站主题

    2024-05-25 · li3huo
    个人网站
    Hugo

    Hugo 提供了大量可选主题 供我们使用。虽然适配工作有一定成本，但这个付出是值得的。这次总结一下如何快速切换适配好的主题。

    查看 hugo server 命令行工具文档，提供了 --theme 参数，用来指定启动时使用的主题，例如

    hugo server --theme hugo-book

    还可以在构造的时候指定配置： hugo --config all-in-obsidian.yaml

    推荐主题 #

    • PaperMod # 很流行的博客主题
    • Book # 很简约
    • Stack # 个人小站
